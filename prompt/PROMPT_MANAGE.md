@esm.ts 파일에서
@esm.ts (16-163)  이부분의 로직중 ESM_CODE_INFO 코드나 findESMCodes 를 위한 prompt 내용을 DB에서 관리하는 메뉴를 추가하고 DB에 입력된 프롬프트중에서 사용으로 설정된 프롬프트를 사용하도록 프로그램을 수정하고 싶어 

먼저 DB에 관련 내용을 관리하는 페이지를 "프롬프트 관리 메뉴" 기능을 추가해 주고
esm.ts 소스는 이 DB에 입력된 값을 참조하도록 수정해줘

DB에 저장할 항목은 최소한은 
"프롬프트 분류", "회사구분",  "refference_content", "prompt", "사용여부"  항목은 보유하고 있어야해

프롬프트 분류 : 요청등록 프롬프트인지, 인사 프롬프트, 등등 다양한 프롬프트 유형을 저장하는 컬럼
회사구분 : 이 시스템을 사용하는 회사마다 프롬프트를 관리할 예정이므로 회사구분코드
refference_content : ESM_CODE_INFO  에 저장될 참고 md 형식의 내용을 저장 (이 부분은 없을 수도 있음)
prompt : 사용자 입력 프롬프트
