# EAR (ERP AI Powered Request System) 구축 프로젝트 개요

## 1. 프로젝트 정보

| 항목 | 내용 |
|------|------|
| 프로젝트명 | EAR (ERP AI Powered Request System) |
| 구축회사 | SKAX |
| 구축팀 | AI ERP팀 |
| 프로젝트 유형 | SI (System Integration) 프로젝트 |
| 개발 환경 | SAP BTP (Business Technology Platform) |

## 2. 프로젝트 배경

본 프로젝트는 ITSM(IT Service Management) 요청을 AI 기반으로 자동 처리하는 시스템을 구축하기 위해 시작되었습니다. 기존의 수동적인 요청 처리 프로세스를 개선하고, 사용자 친화적인 인터페이스를 통해 효율적인 요청 관리를 제공하는 것을 목표로 합니다.

### 2.1 프로젝트 목적

- **AI 기반 요청 처리**: 자연어 처리를 통한 ITSM 요청 자동 생성 및 처리
- **사용자 경험 개선**: 직관적이고 편리한 요청 등록 및 조회 인터페이스 제공
- **지식 기반 시스템**: RAG(Retrieval-Augmented Generation) 기술을 활용한 지식 관리 및 검색
- **프로세스 자동화**: 인터페이스 자동화를 통한 외부 시스템 연동

### 2.2 주요 특징

- **다중 채팅 모듈 지원**: SKAX Chat Module, SK Networks RAG Module 등 다양한 AI 모듈 지원
- **RAG 기반 지식 관리**: 문서 업로드 및 벡터 검색을 통한 정확한 답변 제공
- **요청 템플릿 시스템**: 방화벽, ESM 등 다양한 요청 유형에 대한 템플릿 제공
- **보안 강화**: XSUAA/IAS 인증, IP 화이트리스트, 입력/출력 보안 필터링

## 3. 시스템 범위

### 3.1 포함 기능

1. **요청 관리**
   - EAR 요청 등록 및 조회
   - ESM 요청 등록
   - 시스템 개선 요청 관리

2. **RAG 관리**
   - 문서 업로드 및 관리
   - 벡터 검색 기반 답변 생성
   - 답변 품질 개선 요청

3. **채팅 시스템**
   - 실시간 AI 채팅
   - 채팅 히스토리 관리
   - 의도 감지 및 템플릿 추천

4. **시스템 관리**
   - 사용자 관리
   - 메뉴 관리
   - 권한 관리
   - IP 화이트리스트 관리
   - 보안 설정 관리

5. **프로세스 관리**
   - 프로세스 시각화
   - 인터페이스 자동화

### 3.2 제외 기능

- 외부 ITSM 시스템과의 실제 연동 (인터페이스 자동화 기능은 포함)
- 배치 작업 스케줄링
- 실시간 알림 시스템

## 4. 기술 스택

### 4.1 Frontend

- **프레임워크**: React 18
- **언어**: TypeScript
- **빌드 도구**: Vite
- **라우팅**: React Router v6
- **UI 컴포넌트**: Lucide React (아이콘)
- **차트/시각화**: ReactFlow (@xyflow/react)

### 4.2 Backend

- **런타임**: Node.js 18.x / 20.x / 22.x
- **프레임워크**: Express.js
- **언어**: TypeScript
- **인증**: JWT, XSUAA, SAP IAS
- **HTTP 클라이언트**: Axios

### 4.3 Database

- **운영 환경**: SAP HANA Database
- **개발 환경**: PostgreSQL (선택 가능)
- **벡터 검색**: pgvector (PostgreSQL) / HANA Vector Engine (HANA)

### 4.4 AI/ML

- **LLM**: OpenAI GPT-4o-mini
- **임베딩 모델**: OpenAI text-embedding-3-large
- **RAG 프레임워크**: LangChain

### 4.5 인프라

- **플랫폼**: SAP BTP (Cloud Foundry)
- **스토리지**: SAP BTP Object Store (S3 호환)
- **인증 서비스**: SAP Identity Authentication Service (IAS)
- **권한 관리**: XSUAA (XS User Account and Authentication)
