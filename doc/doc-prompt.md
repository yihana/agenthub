#프로젝트 산출물 작성 요청

## 배경
현재 프로젝트는 1인 개발자 로 진행하여 Cursor 기반으로 개발을 진행했는데 
이제 베트남 협력업체 FPT 와 협력을 준비해야 해서 
현재까지 한 내용을 SI 프로젝트 형식의 산출물로 만들어야해

## 지시사항
현재까지 만들어진 소스코드와 Prompt 디렉토리에 md 파일등을 바탕으로 
SI 산출물을 만들어줘

산출물은 한글과 영문 2가지 버전으로 존재해야해 
산출물 위치는  
doc 폴더 하위에
korean 디렉토리 : 한글버전 산출물
english : 영문버전 산출물

산출물은 일단 md 파일로 만들어 주되, 아키텍처 등은 mermaid 문법을 활용해줘
